# Mise en place du thème

### 1) Renommer le thème et le Text Domain `custom_theme`

### 2) Installer les composants et dépendances :

```sh
yarn install
```

### 2) Compilation et concaténation des fichiers :

```sh
yarn build
```

Pour le watch :

```sh
yarn watch
```
